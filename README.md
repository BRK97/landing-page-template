## Prerequisites
- NodeJS and NPM

# Instructions
1. Clone the repo and install dependencies
  ```
      git clone 
      cd landing-page-template
      npm i
  ```
   
2. Start the server

   To run in development mode
   
   ```
   npm run dev 
   ```
